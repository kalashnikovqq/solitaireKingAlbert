## Прецедент "Выложить карты в расклад"  

| | Выложить карты в расклад |
|---|---|
| Название | Раскладка карт |
| Описание | Система выкладывает карты из колоды в девять колонок расклада и формирует бельгийский резерв в соответствии с правилами игры по указанию игрока. |
| Основной исполнитель | Система |
| Предусловия | Игрок начал новую игру |
| Постусловия | Карты выложены в расклад из 9 колонок и сформирован бельгийский резерв из 7 карт согласно правилам. |

**Основной сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| 1.Игрок инициирует новую игру. | 2.Система принимает команду начала игры. |
| | 3. Система сдает 1 карту в первую колонку расклада. |
| | 4. Система сдает 2 карты во вторую колонку расклада. |
| ... | ... |
| | 11. Система сдает 9 карт в девятую колонку расклада. |
| | 12. Система формирует бельгийский резерв из оставшихся 7 карт (все карты открыты сразу). |
| | 13. Расклад из 9 колонок сформирован. |


## Прецедент "Переложить карту из одной колонки расклада в другую"

| | Переложить карту из одной колонки расклада в другу |
|---|---|
| Название | Перекладывание карты | 
| Описание | Игрок перекладывает карту из одной колонки расклада в другую в соответствии с правилами. |
| Основной исполнитель | Игрок |
| Предусловия | В расклад выложены карты. Есть возможные ходы согласно правилам игры. |
| Постусловия | Карта перемещена из одной колонки в другую в соответствии с правилами. |

**Основной сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| 1. Игрок выбирает карту для перемещения. | 2. Система отмечает выбранную карту. |
| 3. Игрок выбирает колонку, куда переместить карту. | 4. Система проверяет правомерность хода. |
| | 5. Система перемещает карту в выбранную колонку. |

**Альтернативный сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| 4a. Игрок пытается переместить карту в недопустимую позицию. | 5a. Система оставляет карту на месте. |
| | 6a. Система выводит сообщение о недопустимом ходе. |

## Прецедент "Перенос карты в базовые"  

| | Перенос карты в базовые |
|---|---|
| Название | Перенос карты в базовые |
| Описание | Игрок может переместить карту из поля игры в базовые, если она соответствует правилам переноса. |
| Основной исполнитель | Игрок |
| Предусловия | Партия начата. На поле игры есть карта, которую игрок хочет переместить в базовые. |
| Постусловия | Карта перемещена в базовые. |

**Основной сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| 2. Игрок выбирает карту на поле, которую хочет переместить в базовые. | 1. Система ожидает выбора карты. |
| 3. Игрок переносит выбранную карту в базовые в соответствии с правилами. | 4. Система проверяет возможность перемещения карты и ее соответствие правилам. |
| | 5. Система перемещает карту в базовые. |

**Альтернативный сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| 3а. Игрок пытается перенести выбранную карту в базовые, но это противоречит правилам. | 4а. Система обнаруживает ошибку в ходе игрока. |
| | 5а. Система выводит сообщение об ошибке. |
| | 6а. Система оставляет карту на месте. |
| | 7а. Система возвращается к ожиданию хода игрока. |

## Прецедент "Взять карту из резерва"

| | Взять карту из резерва |
|---|---|
| Название | Взятие карты из резерва |  
| Описание | Игрок берет одну из открытых карт из бельгийского резерва для хода. |
| Основной исполнитель | Игрок |
| Предусловия | В резерве есть карты. |
| Постусловия | Одна карта взята игроком из резерва. |

**Основной сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| 1. Игрок выбирает карту из резерва. | 2. Система отмечает выбранную карту. |
| 3. Игрок указывает, куда переместить карту. | 4. Система проверяет правомерность хода. |
| | 5. Система перемещает выбранную карту из резерва в указанное игроком место в расклад. |

**Альтернативный сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| 3а. Игрок пытается переместить карту из резерва в недопустимую позицию. | 4а. Система оставляет карту в резерве. |
| | 5а. Система выводит сообщение о недопустимом ходе. |


## Прецедент "Победа игрока"

| | Победа игрока |
|---|---|
| Название | Выигрыш партии | 
| Описание | Игрок выигрывает партию, выполнив условия победы в пасьянсе. |
| Основной исполнитель | Система |
| Предусловия | Партия начата. Есть возможные ходы. |
| Постусловия | Игрок объявлен победителем. Партия завершена. |

**Основной сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| 1. Игрок делает ходы в соответствии с правилами. | 2. Система отслеживает состояние игры. |
| 3. Игрок размещает все карты по мастям в порядке возрастания на базах. | 4. Система определяет, что условия победы достигнуты. |
| | 5. Система объявляет игрока победителем. |
| | 6. Система предлагает начать новую игру. |

## Прецедент "Проигрыш игрока"

| | Проигрыш игрока |
|---|---|
| Название | Проигрыш партии |
| Описание | Игрок решает, что партия проиграна, если не может совершить дальнейшие правильные ходы или принимает решение завершить игру досрочно. |
| Основной исполнитель | Игрок |
| Предусловия | Партия начата. |
| Постусловия | Игрок объявлен проигравшим. Партия завершена. |

**Основной сценарий:**

| Действия игрока | Реакция системы |
|---|---|
| 1. Игрок делает ходы в соответствии с правилами. |  |
| 2. Игрок осознает, что дальнейшая игра невозможна или принимает решение завершить игру. |  |
| | 3. Система ожидает решения игрока. |
| 4. Игрок объявляет, что партия проиграна. | 5. Система завершает игру и объявляет игрока проигравшим. |
| | 6. Система предлагает начать новую игру. |
