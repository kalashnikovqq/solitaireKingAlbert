## Прецедент "Выложить карты в расклад"  

| | Выложить карты в расклад |
|---|---|
| Название | Раскладка карт |
| Описание | Система выкладывает карты из колоды в девять колонок расклада и формирует бельгийский резерв в соответствии с правилами игры по указанию игрока. |
| Основной исполнитель | Система |
| Предусловия | Игрок начал новую игру |
| Постусловия | Карты выложены в расклад из 9 колонок и сформирован бельгийский резерв из 7 карт согласно правилам. |

**Основной сценарий:**

| Действия исполнителя | Действия системы |
|---|---|
| Игрок инициирует новую игру. | Система принимает команду начала игры. |
| | Система сдает 1 карту в первую колонку расклада. |
| | Система сдает 2 карты во вторую колонку расклада. |
| ... | ... |
| | Система сдает 9 карт в девятую колонку расклада. |
| | Система формирует бельгийский резерв из оставшихся 7 карт (все карты открыты сразу). |
| | Расклад из 9 колонок сформирован. |


## Прецедент "Переложить карту из одной колонки расклада в другую"

| | Переложить карту из одной колонки расклада в другу |
|---|---|
| Название | Перекладывание карты | 
| Описание | Игрок перекладывает карту из одной колонки расклада в другую в соответствии с правилами. |
| Основной исполнитель | Игрок |
| Предусловия | В расклад выложены карты. Есть возможные ходы согласно правилам игры. |
| Постусловия | Карта перемещена из одной колонки в другую в соответствии с правилами. |

**Основной сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| Игрок выбирает карту для перемещения. | Система отмечает выбранную карту. |
| Игрок выбирает колонку, куда переместить карту. | Система проверяет правомерность хода. |
| | Система перемещает карту в выбранную колонку. |

**Альтернативы:**

| Действия исполнителя | Реакция системы |
|---|---|
| Игрок пытается переместить карту в недопустимую позицию. | Система оставляет карту на месте. |
| | Система выводит сообщение о недопустимом ходе. |

## Прецедент "Взять карту из резерва"

| | Взять карту из резерва |
|---|---|
| Название | Взятие карты из резерва |  
| Описание | Игрок берет одну из открытых карт из бельгийского резерва для хода. |
| Основной исполнитель | Игрок |
| Предусловия | В резерве есть карты. |
| Постусловия | Одна карта взята игроком из резерва. |

**Основной сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| Игрок выбирает карту из резерва. | Система отмечает выбранную карту. |
| Игрок указывает, куда переместить карту. | Система проверяет правомерность хода. |
| | Система перемещает выбранную карту из резерва в указанное игроком место в расклад. |

**Альтернативы:**

| Действия исполнителя | Реакция системы |
|---|---|
| Игрок пытается переместить карту из резерва в недопустимую позицию. | Система оставляет карту в резерве. |
| | Система выводит сообщение о недопустимом ходе. |


## Прецедент "Победа игрока"

| | Победа игрока |
|---|---|
| Название | Выигрыш партии | 
| Описание | Игрок выигрывает партию, выполнив условия победы в пасьянсе. |
| Основной исполнитель | Система |
| Предусловия | Партия начата. Есть возможные ходы. |
| Постусловия | Игрок объявлен победителем. Партия завершена. |

**Основной сценарий:**

| Действия исполнителя | Реакция системы |
|---|---|
| Игрок делает ходы в соответствии с правилами. | Система отслеживает состояние игры. |
| Игрок размещает все карты по мастям в порядке возрастания на базах. | Система определяет, что условия победы достигнуты. |
| | Система объявляет игрока победителем. |
| | Система предлагает начать новую игру. |

## Прецедент "Проигрыш игрока"

| | Проигрыш игрока |
|---|---|
| Название | Проигрыш партии |
| Описание | Игрок проигрывает партию, если не может совершить дальнейшие правильные ходы или решает завершить игру досрочно. |
| Основной исполнитель | Система |
| Предусловия | Партия начата. |
| Постусловия | Игрок объявлен проигравшим. Партия завершена. |

**Основной сценарий 1:**

| Действия исполнителя | Реакция системы |
|---|---|
| Игрок делает ходы в соответствии с правилами. | Система отслеживает состояние игры. |
| Положение карт не позволяет сделать правильный ход. | Система определяет, что дальнейшая игра невозможна. |
| | Система объявляет игрока проигравшим. |
| | Система предлагает начать новую игру. |

**Основной сценарий 2:**

| Действия исполнителя | Реакция системы |
|---|---|
| Игрок решает завершить игру досрочно. | Система фиксирует текущее состояние. |
| | Система объявляет игру проигранной. |
| | Система предлагает начать новую игру. |
